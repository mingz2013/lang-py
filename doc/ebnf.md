## EBNF

语言设计采用EBNF语法设计。

|EBNF元符号|含义                |
|:--------|:-------------------|
| ::=     | 定义为，推导为       |
| &#124;  | 或                 | 
| {}      | 含0次在内任意多次重复 |
| []      | 含0次和1次          |
| ()      | 括号内看作一项       |
| .       | 一条生成规则的结束   |
| <>      | 非终结符            |
| ""      | 终结符              |
